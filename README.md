E-commerce UI is one of the most demanding frontend challenges — every pixel affects whether someone clicks "buy." I built Trendverse to push my vanilla JS and CSS further, specifically around layout precision, hover states, and the kind of micro-interactions that make a shopping experience feel premium rather than templated.

What made it hard

Achieving a high-end feel without a framework. When you can't lean on component libraries, every detail is a deliberate choice. That constraint made me a sharper frontend developer.
